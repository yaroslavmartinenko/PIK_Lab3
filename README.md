# Лабораторна робота 3 — Qt Framework

## Варіант 1. Форма опитування (Ігрова тематика)

Qt-застосунок для проведення опитування на ігрову тематику з валідацією числових полів та збереженням результатів у файл.

---

## Функціональність

- 8 запитань ігрової тематики (4 текстових, 4 числових)
- Валідація числових полів через `QIntValidator` з перевіркою діапазону
- Live-підсвітка помилок під кожним полем
- Збереження відповідей у текстовий файл (`%USERPROFILE%\SurveyResults\`)
- Кнопка очищення форми
- Темна кольорова схема (Catppuccin Mocha)

---

## Збірка та запуск

### Qt Creator
1. Відкрити `survey.pro` у Qt Creator
2. Обрати kit **Desktop Qt 6.x MinGW**
3. **Build** `Ctrl+B` → **Run** `Ctrl+R`

### Командний рядок
```bash
qmake survey.pro
mingw32-make
```
